# Agent Skills

This directory contains project-specific Agent Skills for the Prompt Library. Keep skills concise, current, and aligned with the local light-DOM component system documented in `CLAUDE.md`.

## Available skills

| Skill | Use when |
|-------|----------|
| `vanilla-js-development` | Implementing plain JavaScript behavior or browser API work. |
| `prompt-library-architecture` | Working on Prompt Library structure, data flow, prompt rendering, or admin/public app behavior. |
| `prompt-image-creator` | Creating or refreshing prompt preview images in the house visual style. |

## Maintenance rules

- Remove skills that describe deprecated systems or completed migrations.
- Prefer `CLAUDE.md` as the source of truth for styling and component rules.
- Do not add root-level status reports or handoff docs for temporary agent work.
